# Kochi Urban Mobility Transit Optimizer

An end-to-end urban mobility analytics platform for optimizing multimodal transit in Kochi using traffic, weather, and public transport data.

---

## Project Overview

This project aims to analyze urban mobility patterns in Kochi by combining multiple real-world data sources, including:

- 🚗 TomTom Traffic API
- 🌦️ Open-Meteo Weather API
- 🚌 Public transport (GTFS)
- 🗺️ OpenStreetMap road network

The objective is to understand how weather and traffic influence travel times and provide data-driven insights for urban transportation planning.

---

## Tech Stack

- Python
- PostgreSQL + PostGIS
- SQLAlchemy
- Pandas
- Git & GitHub
- Apache Airflow (planned)
- Power BI / Tableau (planned)
